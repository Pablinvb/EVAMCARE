# Registro inicial de riesgos

Este documento inicia el análisis; no sustituye un proceso ISO 14971.

| Peligro | Daño posible | Control requerido |
|---|---|---|
| Falso negativo | Retraso en consulta o tratamiento | Uso profesional, alta sensibilidad, advertencias y derivación |
| Falso positivo | Ansiedad, consulta o tratamiento innecesario | Especificidad mínima, confirmación clínica y lenguaje no concluyente |
| Sesgo por fototipo | Peor atención en pieles subrepresentadas | Muestreo estratificado, métricas por grupo y umbrales de equidad |
| Mala iluminación | Clasificación incorrecta | Protocolo de captura y rechazo automático validado |
| Filtros o maquillaje | Señales visuales alteradas | Detección/rechazo y capacitación del usuario |
| Rostro equivocado | Resultado asignado a otra persona | Un rostro por captura e identificación clínica segura |
| Recomendación terapéutica incorrecta | Irritación o interacciones | No recomendar medicamentos en la primera finalidad |
| Cambio silencioso del modelo | Desempeño no validado | Model registry, firma de artefactos y control de versiones |
| Acceso a fotografías | Daño de privacidad biométrica | Minimización, cifrado, acceso por rol y eliminación verificable |
| Automatización excesiva | Profesional ignora contexto clínico | Interfaz de apoyo, incertidumbre visible y capacitación |
| Fallo del servicio | Interrupción de atención | Modo seguro, monitoreo y procedimiento de contingencia |
| Uso fuera de población | Resultado no confiable | Etiquetado, exclusiones técnicas y registro de uso |
